# Schedulix-simulator
A modular CPU Scheduling Simulator built in C++ to demonstrate and compare scheduling algorithms through performance metrics and execution visualization.

This project is a C++ based simulator designed to help understand how different CPU scheduling algorithms work in real-time.
It allows users to enter process details and analyze the behavior of scheduling algorithms such as FCFS, SJF, Round Robin, and Priority Scheduling.
The simulator calculates key performance metrics like waiting time and turnaround time, making it easier to compare algorithm efficiency.
This project was developed as part of a Design and Analysis of Algorithms (DAA) course.
### Features
- Multiple scheduling algorithms
- Performance metric calculation
- Modular design
- Easy-to-use interface
